UI design project focusing on user experience and responsive design. Utilizes JavaScript to implement resposivness and basic interactivity. Specifically, JavaScript is used to load the main content onto the template page, imlpement navigation and implement design changes during window resizing and usage of the search bar.

    • Chrome 21
    • Edge 79
    • Safari 6.1
  • Mobile:
    • Chrome 91
    • Safari 7
    • Android 4.4
